
# Lab 04 - User Personas

## Persona 1: Amina – First-Year International Student

### 1. Persona Name

**Amina, First-Year International Student**

### 2. User Type

New international student

### 3. Background and Context

**Where does this user experience the problem?**

* While searching for dormitory information, cafeteria options, transportation details, and campus facilities.
* On university websites, social media groups, and student chat groups.

**When does the problem usually happen?**

* Before arriving on campus.
* During orientation week.
* Throughout the first semester while adjusting to university life.

**What digital tools does this user already use?**

* University website
* Facebook groups
* Instagram
* WhatsApp/LINE group chats
* Google Search

### 4. Goals

**What does the user want to achieve?**

* Adapt to campus life quickly.
* Find reliable information about housing, food, and student services.
* Make informed decisions about dormitories and cafeterias.

**What would make the situation easier or faster for the user?**

* A single platform containing trusted campus information.
* Student reviews and recommendations.
* Multi-language support.
* Easy-to-use search and navigation features.

### 5. Pain Points from Lab 03 Evidence

| Pain Point                           | Evidence ID / Respondent | Explanation                                 |
| ------------------------------------ | ------------------------ | ------------------------------------------- |
| Information scattered across sources | R001, R005, R007         | Must search many websites and chats         |
| Language barriers                    | R008, R013, R016         | Some information is difficult to understand |
| Doesn't know where to start          | R002, R014               | New to the university environment           |

### 6. Current Alternatives / Workarounds

| Current Alternative | Weakness / Limitation                   | Evidence                     |
| ------------------- | --------------------------------------- | ---------------------------- |
| Social media        | Information is unverified and scattered | R002, R007, R010, R013, R017 |
| Group chats         | Hard to search and incomplete           | R005                         |
| Asking friends      | May be inaccurate or biased             | R001, R015                   |

### 7. Design Implications

**What should the system/app/platform support?**

* Multi-language support for international students.
* Centralized campus information system.
* Student reviews and verified recommendations.
* Beginner-friendly navigation.

**What should the team avoid?**

* Scattered or duplicated information.
* Complex interfaces with high learning curves.
* Unverified or outdated content.

**Which user need is most important for the MVP?**

* A single trusted platform with clear and reliable campus information.
